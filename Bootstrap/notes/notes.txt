 Bootstrap Grid System 
 ----------------------
 each row -> 12  cols 
 
 container / container-fluid
   -> row 
        -> col-<screen>-<count>
		ex: col-sm-4 , col-md-3
		
		Responsiveness -> website -> all devices
		sm -> mobile
		md -> Tablet
		lg -> Laptop
		xl -> desktop
		xxl -> large-desktop
